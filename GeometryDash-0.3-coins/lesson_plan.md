# Урок 5: Geometry Dash — Добавляем фон и монеты

## Цель урока

На этом уроке мы добавим в нашу игру два новых элемента:
1. **Фоновое изображение** — чтобы игра выглядела красиво
2. **Монеты** — объекты, которые можно собирать

---

## Часть 1: Добавляем фон

### Что мы делаем?

Пока наша игра выглядит скучно — просто серый фон. Давайте добавим красивую картинку на задний план!

### Как это работает?

В Godot есть узел **Sprite** — это просто картинка. Мы уже работали со спрайтами (игрок, враг). Теперь сделаем то же самое, но для фона.

### Пошаговая инструкция

1. Открой сцену `LevelTemplate.tscn`
2. В дереве узлов найди корневой узел `LevelTemplate`
3. Нажми правой кнопкой → **Add Child Node**
4. Выбери **Sprite** и нажми **Create**
5. В инспекторе справа найди свойство **Texture**
6. Перетащи в это поле файл `background.png` из папки `assets`
7. Найди свойство **Z Index** и поставь **-10**

### Что такое Z Index?

Представь, что все объекты — это прозрачные листы бумаги. Z Index определяет, какой лист лежит сверху, а какой снизу.

- **Z Index = 0** (по умолчанию) — обычный слой
- **Z Index = -10** — самый нижний слой (фон)
- **Z Index = 10** — самый верхний слой

Поэтому фон с Z Index = -10 будет позади всех остальных объектов!

### Готовый код

```
Узел: Sprite (дочерний к LevelTemplate)
  Texture: res://assets/background.png
  Position: Vector2(0, 0)
  Z Index: -10
```

## Часть 2: Добавляем монеты

### Что мы делаем?

Создадим объект "монета", который игрок сможет собирать. Когда игрок касается монеты — она исчезает!

### Почему это просто?

Вы уже знаете, как работают **Area2D** — это зоны, которые реагируют на вход объектов. Монета — это просто Area2D со спрайтом!

### Пошаговая инструкция

#### Шаг 1: Создаём сцену монеты

1. В Godot нажми **Scene → New Scene**
2. Добавь корневой узел **Area2D** (назови его `Coin`)
3. Добавь дочерний **Sprite** — перетащи `coin.png` в Texture
4. Добавь дочерний **CollisionShape2D** — выбери **CircleShape2D** с радиусом 24

#### Шаг 2: Настраиваем коллизию

В инспекторе для узла `Coin` (Area2D):
- **Collision Layer** = 0 (монета ни на каком слое)
- **Collision Mask** = 1 (реагирует на игрока — слой 1)

#### Шаг 3: Подключаем сигнал

1. Выбери узел `Coin`
2. Справа перейди во вкладку **Node** (рядом с Inspector)
3. Найди сигнал **body_entered**
4. Нажми **Connect** → выбери метод `_on_Coin_body_entered`
5. Нажми **Connect**

#### Шаг 4: Пишем скрипт

Создай скрипт `Coin.gd` и добавь:

```gdscript
extends Area2D

func _on_Coin_body_entered(body: Node) -> void:
    queue_free()
```

**Что делает этот код?**
- `extends Area2D` — монета наследуется от Area2D
- `_on_Coin_body_entered` — функция вызывается, когда что-то входит в зону монеты
- `queue_free()` — удаляет монету из игры

Это та же функция, которую враг использует при смерти!

#### Шаг 5: Размещаем монеты в уровне

1. Сохрани сцену монеты как `Coin.tscn` в папку `src/Actors`
2. Открой `LevelTemplate.tscn`
3. Перетащи `Coin.tscn` из дерева файлов в дерево узлов
4. Повтори 3-5 раз, разместив монеты на разных платформах

### Готовый код

```gdscript
# Coin.gd
extends Area2D

func _on_Coin_body_entered(body: Node) -> void:
    queue_free()
```

## Что мы узнали сегодня

1. **Sprite** — узел для отображения картинок
2. **Z Index** — порядок отрисовки объектов (меньше = глубже)
3. **Area2D** — зона, которая реагирует на вход объектов
4. **Сигнал body_entered** — вызывается, когда физическое тело входит в зону
5. **queue_free()** — удаляет объект из игры
